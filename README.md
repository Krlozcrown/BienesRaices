# Bienes Raíces - Plataforma Web Full Stack (MVC)

Una aplicación web completa y dinámica para la gestión, compra y venta de propiedades inmobiliarias. Este proyecto implementa una arquitectura robusta orientada a objetos, conectando una interfaz de usuario interactiva con un panel de administración y persistencia de datos en una base de datos relacional.

## Características Clave

* **Arquitectura MVC (Modelo-Vista-Controlador):** Separación limpia de la lógica de negocio, las vistas y los datos para garantizar un código mantenible y escalable.
* **Panel de Administración Completo (CRUD):** Interfaz protegida para administradores que permite crear, leer, actualizar y eliminar propiedades y vendedores.
* **Autenticación y Seguridad:** Sistema de inicio de sesión seguro con validación de sesiones, encriptación de contraseñas y consultas preparadas para mitigar inyecciones SQL.
* **Carga y Gestión de Archivos:** Implementación de lógica para la subida, validación y optimización de imágenes de las propiedades directamente al servidor.
* **Diseño Responsivo Moderno:** Interfaz fluida adaptada a dispositivos móviles, tablets y ordenadores utilizando metodologías de CSS moderno.

## Tecnologías Utilizadas

* **Backend:** PHP (Programación Orientada a Objetos - POO)
* **Base de Datos:** MySQL (Modelado relacional y consultas estructuradas)
* **Frontend:** JavaScript Moderno (ES6+), HTML5 Semántico, CSS3 (Sass / SCSS)
* **Seguridad:** Sanitización de datos y hashes seguros para contraseñas.
